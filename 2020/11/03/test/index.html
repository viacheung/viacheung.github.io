Git

1 代码上传Github上

…or create a new repository on the command line （空仓库）

echo "# mogu" >> README.md
git init//初始化产生一个.git
git add README.md(一般是git add .)
git commit -m "first commit"
git branch -M main//这一步最好做一下否则报错
git remote add origin https://github.com/viacheung/mogu.git//复制代码的https
git push -u origin main

…or push an existing repository from the command line （非空仓库）

git remote add origin https://github.com/viacheung/mogu.git
git branch -M main
git push -u origin main

1 解决Failed to connect to github.com port 443: Timed out

git config --global --unset http.proxy//执行该命令再Push

2 若执行git remote add origin******
出现fatal: remote origin already exists.（报错远程起源已经存在。）

1、先输入 git remote rm origin
2、再输入 git remote add origin**************

3 Pycharm上， push代码到GitHub提示：

fatal: unable to access 'https://github.com/panc-test/pythonProject.git/': OpenSSL SSL_read: Connection was aborted, errno 10053

解决方案：

1、网络不稳定，多试几次

2、更改网络认证设置

3、关闭vpn试试？我还没试过

git config http.sslVerify "false
